```bash

https://github.com/TerrestrialTechnologies/Warez-Works.git
```

2. Open 'index.html' in your browser

3. Enjoy the year 2001

## Local Testing 

Because this page loads data from 'courses.txt' it should be served through a local 
web server, or it may not work. 

From the project directory: 

```bash
python3 -m http.server 8000
```
